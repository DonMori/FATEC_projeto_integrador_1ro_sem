# FATEC_projeto_integrador_1ro_sem

# Python Scheduling System for audio-visual sessions

> This program will be run on your command line interface (CMD, Powershell or the CLI used by yout IDE).
---

When run, it creates JSON files for storing variables and inputed values. No SQL databases were used.

You can run this code by:
- Double clicking main.py
- Openning it directly by your terminal, given you are inside the project's directory:
```
python .\main.py
```
